# Complete File Upload System
A full-stack File Upload System built using **React + TypeScript** for the frontend and **Node.js + Express + TypeScript** for the backend. This project demonstrates multiple file uploads, validation, image compression, upload progress tracking, and file download functionality.

---

##  Features

- Upload multiple files at once
- Drag & Drop file upload
- File type validation
- File size limit validation
- Upload progress tracking
- Image compression using Sharp
- Download uploaded files
- Error handling
- REST API with Express
- React + TypeScript frontend

---

## How to Run
backend run on : http://localhost:3001
frontend run on : http://localhost:5173

# API Endpoints

## Upload Files

```
POST /api/upload
```

Uploads multiple files.

---

## Get Uploaded Files

```
GET /api/upload
```

Returns the uploaded file list.

---

## Chunk Upload

```
POST /api/upload/chunk
```
