# Component Test Suite
A React + TypeScript project demonstrating comprehensive component testing using **Vitest** and **React Testing Library**.

## Task

Write comprehensive tests for 5 complex components covering:

- Form Validation
- Async Data Fetching
- User Interactions
- Error States
- Edge Cases

Target: **90%+ Test Coverage**

---
## Technologies Used

- React
- TypeScript
- Vite
- Vitest
- React Testing Library
- @testing-library/user-event
- Jest DOM

---

## Components

### 1. LoginForm

Features:
- Email validation
- Password validation
- Required field validation
- Successful form submission

### 2. UserList

Features:
- Async API fetching
- Loading state
- Error handling
- Display fetched 

### 3. SearchBar

Features:
- Search input
- Search button
- Clear button
- Input state management

### 4. TodoItem

Features:
- Checkbox
- Edit Todo
- Save Todo
- Delete Todo

### 5. ProductCard

Features:
- Product image
- Product price
- Stock status
- Add To Cart button
- Disabled button when unavailable
