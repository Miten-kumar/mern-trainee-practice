# Flexible Tab Component System

## Overview

This project demonstrates a **Flexible Tab Component System** built with **React, TypeScript, Context API, Node.js, and Express.js**. It follows the **Compound Component Pattern** and supports controlled/uncontrolled modes, keyboard navigation, and lazy loading.

---

## Features

- Compound Component Pattern
- Context API for state management
- Controlled Tabs
- Uncontrolled Tabs
- Keyboard Navigation
- Lazy Loading
- TypeScript Support
- REST API Integration
- Responsive UI
